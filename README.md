# Personal website (GitHub Pages)

This repository contains the source of your GitHub Pages website.

## Quick edit guide
- Update text in `index.md` and the other `.md` pages.
- Put images in `assets/img/` and reference them like: `![](/assets/img/yourfile.jpg)`
- Replace `assets/cv.pdf` with an updated CV PDF whenever needed.

## Publish
1. Push to `main`
2. In GitHub: **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main` → Folder: `/(root)`
3. Your site is available at: `https://<username>.github.io`
